#### 5. 如下代码的输出是什么？为什么？
##### 5.a
```shell
>>> def func(a, b=4, c=5):
...     print(a, b, c)
... 
>>> func(1, 2)
```
输出是`1 2 5`。因为函数定义中为参数`b`和`c`指定了默认值，当调用函数时只提供了参数`a`和`b`的值，参数`c`使用了默认值`5`。所以`a`、`b`和`c`的值分别是`1`、`2`和`5`。

##### 5.b
```shell
>>> def func(a, b, c=5):
...     print(a, b, c)
... 
>>> func(1, c=3, b=2)
```
输出是`1 2 3`。因为在调用函数时，`a`的值被位置形式指定为`1`，而`b`和`c`的值被关键字形式分别指定为`2`和`3`。

##### 5.c
```shell
>>> def func(a, *pargs):
...     print(a, pargs)
... 
>>> func(1, 2, 3)
```
输出是`1 (2, 3)`。因为函数定义中使用`*`时，会以元组形式收集所有位置参数，除了第一个参数`a`以外的参数都被收集到`pargs`中，形成一个元组`(2, 3)`。而后`pargs`被以元组的形式打印了出来。

##### 5.d 
```shell
>>> def func(a, b, c=3, d=4):
...     print(a, b, c, d)
... 
>>> func(1, *(5, 6))
```
输出是`1 5 6 4`。因为在调用函数时，`a`的值被位置形式指定为`1`，`*(5, 6)`被解包为两个位置参数，指定了`b`和`c`的值，参数`d`使用了默认值`4`。
