#### 3. 编写基本循环
##### a. 写个for循环，打印字符串`S = "Spam"`中每个字符的ASCII码。使用内置函数`ord`（character）把每个字符转换成ASCII整数（在交互模式下测试来观察其工作方式）。
```shell
>>> S = "Spam"
>>> for c in S:
...     print("'" + c + "'的ASCII码是", ord(c))
... 
'S'的ASCII码是 83
'p'的ASCII码是 112
'a'的ASCII码是 97
'm'的ASCII码是 109
```
`ord`函数会返回一个字符的ASCII码整数值。
在交互模式下输入`for`行的`:`后，按回车键进入循环体时，交互环境会自动给出`...`，提示继续输入循环体的内容，直到给出一个空行结束循环体的输入。

##### b. 接着，修改循环来计算字符串中所有字符的ASCII码的总和。
```shell
>>> asciiSum = 0
>>> for c in S:
...     asciiSum += ord(c)
...
>>> asciiSum
401
```

##### c. 最后，再次修改代码，来返回一个新的列表，其中包含了字符串中每个字符的ASCII码。表达式`map(ord,S)`是否有类似的效果?
```shell
>>> asciis = []
>>> for c in S:
...     asciis.append(ord(c))
... 
>>> asciis
[83, 112, 97, 109]
>>> map(ord, S)
<map object at 0x000001E220FC9240>
```
两者的效果不完全相同，`map`函数会返回一个`map`对象而非一个列表。`map`对象是一个惰性的可迭代对象，避免一次性生成整个`map`转换结果列表，提高代码的运行性能。
可以用`list`函数把`map`对象转换成一个列表：
```shell
>>> list(map(ord, S))
[83, 112, 97, 109]
```
