#### 4. 排序字典
**字典是无序集合体。编写一个for循环来按照排序后（递增）顺序打印字典的项目。提示：使用字典`keys`和列表`sort`方法，或者较新的`sorted`内置函数。（`D = {'f':6, 'c':3, 'a' :1, 'g':7, 'e':5, 'd':4, 'b':2}`）**
```python
D = {'f':6, 'c':3, 'a' :1, 'g':7, 'e':5, 'd':4, 'b':2}

sortedKeys = sorted(D.keys())
for k in sortedKeys:
    print(k, ":", D[k])
```
输出如下：
```shell
a : 1
b : 2
c : 3
d : 4
e : 5
f : 6
g : 7
```
